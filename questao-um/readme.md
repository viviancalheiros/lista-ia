# Shell de Sistema Especialista - Diagnóstico Oftalmológico

Esse projeto consiste em um Sistema Baseado em Conhecimento (SBC), também conhecido como Sistema Especialista, desenvolvido para fazer a simulação de diagnósticos oftalmológicos utilizando os conceitos clássicos de Inteligência Artificial Simbólica. 

## Objetivo

O principal objetivo do sistema é simular o raciocínio clínico de um oftalmologista utilizando uma abordagem computacional declarativa. Através de um questionário dinâmico focado em sintomas (como dor nos olhos, secreção, coceira e idade).

*Nota: Este sistema possui caráter puramente acadêmico e didático, não substituindo uma consulta médica real.*

# Como Funciona a Inteligência Artificial do Sistema?

Diferente da IA moderna baseada em redes neurais e aprendizado estatístico (Machine Learning), este sistema utiliza IA Simbólica Baseada em Regras. Ele é composto por três pilares fundamentais:
- Base de Conhecimento (Fatos e Regras)
- Motor de Inferência Híbrido
- Mecanismo de Explicação (Auditoria de IA)
